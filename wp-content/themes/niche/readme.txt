== Installation ==

1. Primary:
 = Login to your wp-admin account and go to Appearance -> Themes.
 = Select "Install" tab and click on the "Upload" link.
 = Select "niche.zip" and click on "Install Now" button.
 = In case of errors, use an alternate method.
 
2. Alternate:
 = Unzip the template file (niche.zip) that you have downloaded.
 = Upload the entire folder (niche) to your server via FTP and place it in the /wp-content/themes/ folder.
 = Do not change the directory name.
 = The template files should be there now: /wp-content/themes/niche/index.php (example).
 
3. Log into your WP admin panel and click on "Appearance". Go to "Themes" tab.

4. Now click on "niche" to activate the theme.

5. one custom widgets :
	i) niche contact info widget for display contact info in sidebar or footer.
	
6. Theme Customizer	Options:	

	a) Basic Settings:
           You can manage here site logo, favicon, blog title and contact information text.	

	b) Home Page Settings:
		i)		Go to Page --> Add New 
		ii)		Page Attributes --> Select "Home Page".
		iii)	Slider Section	: You can add Soliloque Slider Id.
		iv)		Our path to Perfection : You can set home page title, images, description and background image.
        v)		Blog Section : you can select category of posts and add title, description of posts slider.
		vi)		For set home page : Go to Settings	-->	Front page displays	--> Checed  "A static page (select below)" --> Front page --> select your Page.
                
    c) Social Settings:
            You can add here social media links.

7. How to setup custom menu ?

You can change the content of the top right menu by create your own.

Create your custom menu :

i. In your admin panel, go to Appearance > Menu.
ii. Add content (pages, ...) to the menu and save.

Attach the custom menu to the theme :

i. In your admin panel, go to Appearance > Menu.
ii. Create a Menu and set checked "Top Header Menu" to display header menu.
